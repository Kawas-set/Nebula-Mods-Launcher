🚀 Release

Nebula Mods Launcher helps users quickly find, review, and install Among Us mods from official GitHub releases.

There is no longer any need to manually search for releases, extract archives, or guess which files match your game version.

The launcher shows a catalog of mods, latest versions, release status, author, source, and license information, and helps users choose the correct assets for installation.

It supports both archived mods and DLL-based mods through BepInEx, while remembering your game folder for a smoother workflow.

⚙️ Installation

  • Download the latest release of Nebula Mods Launcher
  • Extract the launcher files to any folder
  • Run the launcher
  • Select your Among Us folder
  • Choose a mod from the catalog
  • Review mod information and assets
  • Install the selected mod

🖥️ Requirements

  • A legitimate copy of Among Us
  • Windows operating system
  • Internet connection for release loading and downloads
  • BepInEx for DLL-based mods when required

📝 Notes

  • Not all mods are compatible with each other
  • Some mods may only support specific game versions
  • Some mods may require additional setup steps outside the launcher
  • Availability depends on the original public source remaining online
  • Mod authors retain ownership of their own releases and materials
